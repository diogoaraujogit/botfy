# Botfy

## Description
MVP's Dashboard that monitors the performance of a support team.

## Tools and Technologies
- React
- SCSS
- Javascript
- Material UI

## Instructions

#### `Run Local`
Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`   

To Start Server:

`npm start`  

To Visit App:

`localhost:3000`
